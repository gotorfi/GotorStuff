RUINED. is a horror title conceived by GotorFI and developed by ShrooMush Studios for the Roblox platform. The game focuses on high realism and authentic horror elements to deliver an immersive player experience.

Development began in 2024, and following a dedicated production cycle, the game was officially released in 2026. To ensure a unique visual identity, the project features over 1,000 custom-made, unique models, setting it apart from standard platform assets.

Created and developed by GotorFI



BaldJumper is a personal development project focused on precision platforming. The core gameplay involves navigating a character through various levels to reach a designated goal. A key technical feature of the project is the built-in level editor, which allows users to design and create their own custom stages.

Technical Specifications:
Language: Python
Library: Pygame
Status: Personal portfolio project (not for commercial sale)

While the game is not publicly distributed for sale, the source material and gameplay can be reviewed and tested by downloading the package via the button below.
